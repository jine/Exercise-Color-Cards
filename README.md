# Exercise-Color-Cards

I denna övning ska ni skriva klart CSS-koden så ni får följande resultat på skärmen:

![Färdig resultat](result.png "Resultatet")

Det finns 3 olika svårigheter. 
- Lätt: easy.css
- Medel: normal.css
- Svår: hard.css

Ni väljer vilken svårighet ni vill testat på , genom att kommentera/avkommentera rätt rad i HTML-filen.
All kod som ni sedan behöver skriva, ska vara i er valda CSS_fil.


Lycka till!
